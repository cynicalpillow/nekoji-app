# nekoji-app
Anime tracking app in development for Android using Anilist API v1 (soon v2).

**FEATURES:**
- Anime Info
- Live countdown to next episode
- Season lists
- Airing anime list (weekly)
- Search
- Sorting, grid/list view toggling
